Spanisches Sprachpaket für Magento Commerce

(Magento Community Modules - Spanish (Spain) Language Pack)

(Paquete de idioma español para Magento Commerce)

Nota importante acerca Magento Versión 1.9

Magento Inc. tiene con la versión 1.9 su propio, semi-automáticamente traducido, paquete de idiomas como preinstalado. El que quiere usar el LocalePack_es_ES, primero debe eliminar la carpeta app/locale/es_ES para que el paquete de idioma se puede instalar.

Important note on Magento Version 1.9

Magento Inc. has pre-installed with the 1.9 private, semi-automatically translated, language pack. Who wants to use the LocalePack_es_ES must first delete the app/locale/es_ES folder so that this language pack can be installed.

Wichtiger Hinweis zur Magento Version 1.9

Magento Inc. hat mit der 1.9 ein eigenes, teilautomatisch übersetztes, Sprachpaket vorinstalliert. Wer das LocalePack_es_ES benutzen möchte, muss vorher den Ordner app/locale/es_ES löschen, damit dieses Sprachpaket installiert werden kann.
